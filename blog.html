Question: Discuss the scope of var, let, and const?

Answer:

  • Var: It's not a block scoped element and it can cause some unexpected issues.
  • Let: It's a block scoped element. You can access it only the scoped areas. You can reassign it's value.
  • Const: It's also a block scoped element like Let. But you can't reassign the value.

Question: Tell us the use cases of null and undefined?

Answer:

  • Null: When you assign a value null, that means the elements has no value or you want to show the value intentionally.
  • Undifined: If can declare a variable but forget to assign the value, it will give you undifined. And trying to access any element or value which doesn't exist, it will also return undifined.

Question: What do you mean by REST API??

Answer:

REST API is like a menu in a restaurant. It tells you what dishes (resources) are available and how you can order (request) them. You use common words like "get," "create," "update," and "delete" to interact with the menu. The waiter (server) then brings you the food (data) you asked for. It's a way for computer programs to talk to each other over the internet, just like people order food at a restaurant.
